## 说明 
本项目参考 `Spring-AI`, 使用 `deepSeek` 模型。

## 启动项目：
1. 启用 `Lombok` 注解处理
2. 增加环境变量 `DEEPSEEK_API_KEY`, 值为从 [DeepSeek](https://deepseek.com/) 获取的 API 密钥。
